## 📌 Project Overview
This project proposes a non-invasive, AI-based approach to predict blood groups using fingerprint images. By leveraging the **Swin Transformer** deep learning architecture, the system analyzes complex ridge patterns to identify blood types (ABO and Rh factor) without traditional laboratory testing.
